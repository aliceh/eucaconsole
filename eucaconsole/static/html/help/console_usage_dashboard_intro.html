Logging Into the Dashboard

This section covers how to login to the Eucalyptus Console and start using the dashboard.

  • Console Login
    This screen allows you to log in to the Eucalyptus console. If you've forgotten your password or don't have login credentials, please contact your system administrator.
  • Navigate the Dashboard
    The dashboard is your starting point for using the Eucalyptus console. From the Dashboard, you can access instances, storage items (volumes, snapshots, and buckets), and networking and security objects (key pairs, security groups, and IP addresses).
