Locating the Console Configuration File

The Eucalyptus Console configuration settings are stored in the console.ini file.

For Centos and RHEL installations from packages, this file is located in /etc/eucalyptus-ui/server/console.ini.

For Ubuntu installations from packages, this file is located in /opt/euca-ui/server/console.ini.

For installations from source, this file is in <your-git-directory>/eucalyptus-ui/console.ini
