Create an Image

This page allows you to create an image. All instances are made from images. For more information on images, see Image Overview in the Eucalyptus Image Management Guide.

Machine

Specify basic information about your image in this section.

  1. Type the name of your new image in the Name text box. Refer to the rules for naming that appears below the text box.
  2. A reference to the image it was created from is pre-populated in the Description. text box for your convenience. However, you can change the description by typing in a different description for the image.

Storage

This section appears here for images associated with non-EBS backed instances and allows you to specify where to store your image. For images associated with EBS-backed instances, refer to the Advanced options for configuring storage information.

  1. Specify a bucket to store your image by selecting one from the Bucket name drop-down list box or entering a name to create a new bucket.
  2. A prefix of image is already assigned for the filename prefix of your image but you can change it by typing in a different prefix in the Prefix text box.
  3. Refer to the next section to add tags. Refer to the Advanced options for configuring additional settings. Otherwise, click the Create Image button to create your image.

Add tags

To help you manage your cloud's instances, images, and other Eucalyptus resources, you can optionally assign your own metadata to resources in the form of tags. You can use tags to create user-friendly names, make resource searching easier, and improve coordination between multiple users. You can optionally add tags by performing the following steps:

To add new tags:
  1. Type the key name for your tag into the name... text box.
    Note: Tags cannot start with "euca:" or "aws:".
  2. Type the value for your tag into the value... text box.
  3. Click the Add Tag button.
  4. If you wish to add additional tags, repeat the preceding steps.

To delete one or more tags:
Move your mouse over the tag you wish to delete and click the X button.

Specify Advanced Options

This panel is only available for images associated with EBS-backed instances. These advanced options allow you to add user data, override the kernel and RAM disk IDs, specify private networking, and add additional storage for your new image.

  1. Check the No reboot check box to keep the instance from shutting down before the image is created.
    Not allowing the instance to shut down before the image is created can impact the file integrity of the image.
  2. For EBS-backed instances, you can configure the root volume or additional storage for your instance in the Storage section:
    1. Change the size of the root volume by entering the size of the attached storage in gigabytes into the Size (GB) text box.
    2. Select the Delete on terminate check box if you want the attached storage deleted when the instance is terminated.
    3. You can configure additional storage for your instance by selecting a volume type from the Volume drop-down list box.
    4. Type the desired mapping for the storage into the Mapping text box (for example: /dev/sdb).
    5. If you want to create the storage from an existing snapshot, specify it in the Create from snapshot drop-down list box.
    6. Type the size of the attached storage in gigabytes into the Size (GB) text box.
    7. Select the Delete on terminate check box if you would like the attached storage to be deleted when the instance is terminated.
    8. Click the Add Device button to add this storage.
      Added storage displays as a row in the table under the Storage area.
    9. You can delete existing storage by clicking the minus button on the right side of the storage list item.
  3. Click the Create Image button to create your new image.

Creation confirmation

A Warning page opens with information about image creation process. For images associated with non-EBS backed instances, a prompt for your password is required to proceed. EBS-backed instances do not require a password.

  1. Enter your password in the Your password text box and be sure have completed all the steps outlined on the page before preoceeding.
  2. To proceed, click the I Understand, Let's Do This! button. Otherwise, click outside the page or click the X control from the top corner of the page to exit.
    If you proceeded to create the image, another page opens, stating that the image is successfully created but it will take a few minutes, then the Image Detail window opens, showing the newly-created image with a status of pending.
  3. Refer to the Help in the Image Detail page for instructions associated with that page.
