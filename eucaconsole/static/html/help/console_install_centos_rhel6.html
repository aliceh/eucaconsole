Install on Centos / RHEL 6.3

To install the Eucalyptus Console from packages on Centos and RHEL 6.3:

  1. Configure the EPEL repository by editing the file /etc/yum.repos.d/epel.repo: 
    [epel]
name=Extra Packages for Enterprise Linux - $basearch
baseurl=http://mirror.eucalyptus/epel/$releasever/$basearch
enabled=1
gpgcheck=0
  2. Create a new file mamed /etc/yum.repos.d/eucalyptus-ui.repo and add the following content: 
    [eucalyptus-ui]
name = Eucalyptus UI
mirrorlist = http://packages.release.eucalyptus-systems.com/api/1/genrepo/?distro=centos&releasever=6&arch=x86_64&url=git://github.com/eucalyptus/eucalyptus-ui.git&ref=master
enabled = 1
gpgcheck = 0
  3. Install dependencies: 
    yum install python-boto python-tornado
  4. Install the package using yum: 
    yum install python-eucalyptus-ui
Your installation is now complete.
You are now ready to configure the Eucalyptus Console.
