Working with Security Groups

This section covers how to work with security groups using the Eucalyptus Console.

  • Manage Security Groups
    This screen allows you to view a list of your security groups and create, modify and delete security groups. You can page through the list of security groups by clicking the navigation buttons at the bottom of the screen.
  • Create a Security Group
    Eucalyptus enables you to control access to your cloud using security groups. A security group contains a group of rules that control inbound traffic to instances in the group for the specified protocols and ports.
  • Edit Security Group
    This dialog allows you to edit a security group.
  • Delete Security Group
    This dialog box allows you to confirm or cancel a key pair delete operation.
